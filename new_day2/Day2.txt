PHP의 연산자

1. 산술 연산자
+(덧셈), -(뺄셈), *(곱셈), /(나눗셈), %(나머지), **(제곱)

2. 대입 연산자
=, +=, -=, *=, /=, %=, .=(연결)

3. 증감 연산자
++, --

4. 비교 연산자
==, ===, !=, !==, <, >, <=, >=

5. 논리 연산자
&&, ||, !

6. 비트 연산자
&, |, ^, ~, <<, >>

7. 삼항 연산자
조건식 ? 반환값1 : 반환값2;
조건식이 true면 반환값1을 저장, false면 반환값2를 저장합니다.

8. 문자열 연산자
PHP에서는 문자열을 연결할 경우 .(점)을 사용합니다.

1. if문
    - if문
        if(조건문) {
            조건문이 true인 경우 실행할 문장;

        }
    - if else문
        if(조건문) {
            조건문이 true인 경우 실행할 문장;
            ...
        }else{
            조건문이 false인 경우 실행할 문장;
        }
    - if else if문 
        if(조건문1){
            조건문1이 true인 경우 실행할 문장;
            ...
        }else if(조건문2){
            조건문2이 true인 경우 실행할 문장;
            ...
        }
        ...
        else{
            모든 조건문이 false인 경우 실행할 문장;
        }

2. switch문

switch(변수) {
    case 값1:
        변수와 case의 값1과 같을 경우 실행할 문장;
        break;
    case 값2:
        변수와 case의 값2와 같을 경우 실행할 문장;
        break;
    ...
    default:
        변수와 case의 값이 모두 일치하지 않는 경우 실행할 문장;
}


반복문

1. while 문

while(조건식){
    조건식의 결과가 true인 동안 반복할 문장;
}

do {
    조건식의 결과가 true인 동안 반복할 문장;
    (조건식이 처음부터 false라도 1번은 무조건 실행)
}while(조건식);


2. for 문

for(변수의 선언 및 초기화; 조건식; 증감식){
    조건식의 결과가 true인 동안 실행할 문장;
}

하나의 특별한 목적의 작업을 수행하도록 설계된 독립적인 블록을 의미합니다.

function 함수명(매개변수1, 매개변수2 ..){

}


내장함수(문자열 함수)

strlen() : 전달 받은 문자열의 길이를 반환합니다.
strcmp() : 전달 받은 두 개의 문자열을 서로 비교합니다. 첫번째 인수의 문자열이 두번째 인수의 문자열보다 크면 양수를, 작으면 음수를 반환합니다. 또한 두 문자열이 같으면 0을 반환합니다.
strstr() : 해당 문자열에서 전달 받은 문자열과 처음으로 일치하는 부분을 찾습니다. 일치하는 부분이 존재하면 처음으로 일치하는 부분을 포함한 이후 모든 문자를 반환합니다.
strpos() : 해당 문자열에서 전달 받은 문자열과 처음으로 일치하는 부분을 시작 인덱스를 반환합니다.
substr() : 해당 문자열에서 특정 인덱스부터 전달받은 길이만큼의 일부분을 추출하여 반환합니다. 전달받은 인덱스가 양수인 경우에는 해당 문자열의 끝까지를 반환합니다. 만약 전달받은 인덱스가 음수라면 해당 문자열 끝부터 전달받은 음수의 절댓값 만큼의 문자열을 반환합니다.
strtolower() : 전달받은 문자열의 모든 문자를 소문자로 바꿔줍니다.
strtoupper() : 전달받은 문자열의 모든 문자를 대문자로 바꿔줍니다.
explode() : 특정 문자를 기준으로 전달 받은 문자열을 나눠서 하나의 배열로 반환합니다.
str_replace() : 해당 문자열에서 전달 받은 문자열을 모두 찾은 후에, 찾은 문자열을 대체 문자열로 교체합니다.
substr_replace() : 해당 문자열에서 특정 위치의 문자들을 대체 문자열로 교체합니다.


쿠키 
웹 사이트에 접속할 때 서버에 의해 사용자의 컴퓨터에 저장되는 정보를 의미합니다. 웹 사이트는 저장된 사용자의 정보를 클라이언트 측의 컴퓨터에 남겨서 필요할 때마다 재사용합니다. 쿠키의 데이터 형태는 key 와 value로 구성되고 string으로만 이루어져 있습니다. 4kb이상 저장할 수 없습니다.

쿠키를 생성하는 방법
setcookie(쿠키이름, 값, 만료시간, 저장위치, 도메인정보, 프로토콜);

쿠키를 읽어오는 방법
변수이름 = $_COOKIE[쿠키이름];

쿠키를 삭제하는 방법
쿠키 만료시간을 0 또는 과거시간으로 변경해주면 삭제됩니다.

time() 
현재 시간을 반환합니다.

isset()
데이터가 존재하는지 여부(true, false)를 판단합니다.
